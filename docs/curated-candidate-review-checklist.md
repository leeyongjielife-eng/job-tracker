# Curated Candidate Review Checklist

## 用途

这份清单只服务当前 `linkedin_curated_candidates.json` 里的 12 条候选。

目标不是再扩池，而是逐条核实：

- 当前推断摘要是否需要替换为真实卡片摘要
- 岗位是否真的有 AI / agent / model / platform 语境
- 是否应从 `观察` 升级为 `保留`
- 是否应从 `观察` 直接剔除

## 处理顺序

先处理 `高优先`，再处理 `观察`。

## 判断标准

### 升级为保留

- 标题直接命中 `AI / agent / model / gateway / router / platform`
- 或真实摘要明确提到：
  - `AI tools`
  - `agent`
  - `copilot`
  - `model platform`
  - `prompt`
  - `RAG`
  - `multimodal`
  - `inference`
  - `assistant`

### 保持观察

- 标题偏泛，但真实摘要能看出部分 AI / workflow / tooling 语境
- 与你的转型方向有关，但证据仍不够强

### 直接剔除

- 标题和真实摘要都更像普通业务 PM
- 没有清晰 AI / model / agent / platform 语境
- 只是因为推断摘要或泛化词才被留住
